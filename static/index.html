Check Your Internet Security

अपनी इंटरनेट सुरक्षा जांचें

Test your website, email, and connection against modern internet security standards

अपनी वेबसाइट, ईमेल और कनेक्शन को आधुनिक इंटरनेट सुरक्षा मानकों के साथ जांचें

Security Tests

सुरक्षा परीक्षण

Website Test

वेबसाइट परीक्षण

Check if your website meets modern standards for security, including HTTPS, IPv6, DNSSEC, and security headers.

जांचें कि क्या आपकी वेबसाइट HTTPS, IPv6, DNSSEC और सुरक्षा हेडर सहित सुरक्षा के आधुनिक मानकों को पूरा करती है।

Email Test

ईमेल परीक्षण

Verify your email setup for security standards like SPF, DKIM, DMARC, and STARTTLS.

SPF, DKIM, DMARC और STARTTLS जैसे सुरक्षा मानकों के लिए अपने ईमेल सेटअप को सत्यापित करें।

Connection Test

कनेक्शन परीक्षण

Analyze your internet connection for IPv6 and other modern protocols.

IPv6 और अन्य आधुनिक प्रोटोकॉल के लिए अपने इंटरनेट कनेक्शन का विश्लेषण करें।
